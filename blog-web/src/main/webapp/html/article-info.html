获取文章详细信息接口

地址：

/Article/GetArticleInfo

方法：

GET

是否需要登录：

否

参数说明：

参数名称 是否必选 参数类型 参数描述
articleId 是 int 文章id

请求示例：

curl -d 'articleId=1' '/Article/GetArticleInfo'

返回值说明：

参数名称 是否必选 参数类型 参数描述
webUser 是 WebUser 文章用户信息类
webUser.id 是 Integer 文章用户id
webUser.username 是 string 文章用户名称
webUser.nickname 是 string 文章用户昵称
blogConfig 是 BlogConfig 用户博客配置信息类
blogConfig.id 是 Integer 用户博客id
blogConfig.introduction 是 string 用户个人介绍
blogConfig.address 是 string 用户博客地址
blogConfig.blogTitle 是 string 用户博客标题
blogConfig.blogSubTitle 是 string 用户博客小标题
categoryName 是 CategoryName 文章分类信息类
categoryName.id 是 Integer 文章分类id
categoryName.categoryName 是 string 文章分类名称
userArticle 是 UserArticle 文章信息类
userArticle.title 是 string 文章标题
userArticle.content 是 string 文章内容
userArticle.readTimes 是 Integer 文章阅读次数
userArticle.thumbupTimes 是 Integer 文章点赞次数
userArticle.commentTimes 是 Integer 文章评论条数
userArticle.createTime 是 string 文章创建时间
userArticle.updateTime 是 string 文章上次更新时间
