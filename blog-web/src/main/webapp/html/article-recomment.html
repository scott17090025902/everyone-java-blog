获取评论的回复接口

地址：

/Article/GetMoreReComment

方法：

GET

是否需要登录：

否

参数说明：

参数名称 是否必选 参数类型 参数描述
commentId 是 int 评论id
offset 是 int 分页开始头回复id
size 是 int 加载回复个数

请求示例：

curl -d 'commentId=1&offset=3&size=10' '/Article/GetMoreReComment'

返回值说明：

参数名称 是否必选 参数类型 参数描述
webUser 是 WebUser 回复用户信息类
webUser.id 是 Integer 回复用户id
webUser.username 是 string 回复用户名称
webUser.nickname 是 string 回复用户昵称
parentUser 是 WebUser 回复父级用户信息类
parentUser.id 是 Integer 回复父级用户id
parentUser.username 是 string 回复父级用户名称
parentUser.nickname 是 string 回复父级用户昵称
articleComment 是 ArticleComment 回复评论信息类
articleComment.id 是 Integer 评论id
articleComment.createTime 是 string 评论创建时间
articleComment.updateTime 是 string 评论上次更新时间
articleComment.comment 是 string 评论内容
articleComment.parentsCommentId 是 string 父级评论路径
