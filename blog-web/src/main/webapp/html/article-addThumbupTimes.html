添加文章的一次点赞接口

地址：

/Article/AddThumbupTimes

方法：

POST

是否需要登录：

否

参数说明：

参数名称 是否必选 参数类型 参数描述
articleId 是 int 文章id

请求示例：

curl -d 'articleId=1' '/Article/AddThumbupTimes'

返回值说明：

参数名称 是否必选 参数类型 参数描述
flag 是 boolean 是否成功标志

返回示例：

{"code":200,"message":"请求成功","data":true}
